# [PR 233](https://github.com/vig-os/devcontainer/pull/233) feat(remote): Tailscale key injection and --open/--yes flags

## Summary

- Add `--open cursor|code|none` flag to decouple IDE from infra (`--open none` for headless/scripted use)
- Add `--yes` flag for non-interactive mode
- Add `inject_tailscale_key()` — generates ephemeral Tailscale auth keys via OAuth API and injects into remote `docker-compose.local.yaml`
- Key injection is fully opt-in: silent no-op when `TS_CLIENT_ID`/`TS_CLIENT_SECRET` are absent

## How it works

```
Local: TS_CLIENT_ID + TS_CLIENT_SECRET
  → OAuth token from api.tailscale.com
  → Ephemeral auth key (tag:devcontainer)
  → SSH inject into remote docker-compose.local.yaml
  → compose up → container joins tailnet
```
