# [PR 406](https://github.com/vig-os/devcontainer/pull/406) fix(ci): remove sync-main-to-dev workflow_dispatch CI trigger

## Description

Removes the postŌĆōsync-PR `workflow_dispatch` step from `sync-main-to-dev` in this repo and in the workspace template. That dispatch did not appear in the PR status check rollup, so it could not satisfy branch protection ([issue #405](https://github.com/vig-os/devcontainer/issues/405) analysis). Also drops `actions: write` from the sync job, which was only needed to run workflows via the API.
