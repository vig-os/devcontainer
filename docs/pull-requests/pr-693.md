# [PR 693](https://github.com/vig-os/devcontainer/pull/693) fix(ci): make install_workspace a staticmethod fixture (#691)

## Description

Resolves the `PytestRemovedIn10Warning` emitted by the install-script integration suite. The class-scoped `install_workspace` fixture was defined as an instance method — a pattern pytest 9 deprecates and pytest 10 **removes**. Since the project pins `pytest==9.1.1`, a future Renovate/Dependabot bump to pytest 10 would error at collection and take out the entire `TestInstallScriptIntegration` suite. This converts the fixture to a `@staticmethod` (it never used `self`), keeping the class-scope "run `install.sh` once per class" behaviour.
