# [PR 377](https://github.com/vig-os/devcontainer/pull/377) feat: centralize shared git and release recipes in justfile.gh

## Description

Centralizes shared downstream helper recipes in `justfile.gh` so GitHub-oriented and constant recipes are not duplicated in project-level files. The PR moves release and git helpers to a single source, keeps root behavior via import, updates template outputs, and adds focused integration coverage for template recipe exposure.
