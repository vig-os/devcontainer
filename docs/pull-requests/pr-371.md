```diff
     - name: Sync Python dependencies
       if: inputs.sync-dependencies == 'true'
       shell: bash
-      run: uv sync --frozen --all-extras
+      run: |
+        set -euo pipefail
+
+        set +e
+        uv sync --frozen --all-extras
+        rc=$?
+        set -e
+
+        if [ "$rc" -ne 0 ]; then
+          echo "WARNING: uv sync failed (exit $rc), clearing cache and .venv before retry..."
```

The `set +e` / `set -e` toggling to capture `uv sync`'s exit code is more brittle than necessary (future edits could accidentally run additional commands while `-e` is disabled). In bash, `set -e` does not trigger an exit for commands in an `if ! ...; then` condition, so this can be simplified while still capturing the real exit code and keeping strict mode enabled throughout.
