# [PR 332](https://github.com/vig-os/devcontainer/pull/332) ci: enforce downstream release validation gate

## Description

Implements an automated cross-repository validation gate for release publication, including upstream gating logic, downstream dispatch/release artifact handling, and follow-up hardening from Copilot review comments.
Also refactors release documentation to keep smoke-test gate details in a dedicated document and removes the workspace-level dispatch workflow so dispatch handling is owned by the smoke-test template.

- `.github/workflows/release.yml`
