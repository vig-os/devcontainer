# [PR 668](https://github.com/vig-os/devcontainer/pull/668) build(nix): decommission the Debian path — Nix is the only build (#642)

## Summary

Completes the migration: **Nix is the only devcontainer image build**; the Debian/Containerfile path is removed entirely. Rollback, if ever needed, is to branch from the last Debian-built tagged release.

### Removed / Nix-only
- **Deleted** `Containerfile`, `scripts/build.sh`, `scripts/prepare-build.sh`, `.hadolint.yaml` (+ the synced template copy).
- **`build-image` action** is Nix-only (`nix build .#devcontainerImage` → tar); dropped the `builder` selector, `type=gha` cache, Docker Hub login + buildx. **`release.yml`** drops the `builder` input (Nix always). **`ci.yml`** build now produces the Nix image, so its **Image Tests + Integration Tests run on the Nix image**.
- **`security-scan.yml`**: removed the Debian `scan-latest` Trivy job; `scan-nix-image` (vulnix + SBOM, blocking via `vulnix-gate`) is the sole scanner.
- **`ci.yml` PR Trivy scan → non-blocking** awareness + SBOM (apt scanner is dark on Nix; the nightly vulnix gate is authoritative).
- **`.trivyignore`**: removed the ~78 Debian OS-package entries (kept the image-agnostic `gh`/jwt entries). **`renovate.json`**: dropped the `dockerfile` manager. **pre-commit**: removed the hadolint hook (hadolint stays available via the flake). Removed dead hadolint installs from `setup-env`/`test-project`/`requirements.yaml`/manifest.
- **`justfile`**: local `just build` now builds + loads the Nix image (`:dev`); `just test-image`/`test-integration` run against it.
- Scrubbed Debian/apt remnants from `docs/CONTAINER_SECURITY.md`.

## Verified locally
- `nix build .#devcontainerImage` + `just build` load the `:dev` image into podman; `test_image.py` is 63/63 (from #666).
- `check-expirations .trivyignore .vulnixignore` → 29 exceptions validated.

## Note / risk
- This is the **first time the integration suite runs on the Nix image** (`test-suite=all` dispatched). Any residual install/scaffold gaps surface here — I'll iterate to green.

## Dependencies
- Depends-on: #639 + #666 (✅ merged). Completes the epic's Nix-only build path.
