# [PR 45](https://github.com/vig-os/devcontainer/pull/45) feat: add --org flag to install script (#33)

## Description

Add `--org` flag to `install.sh` allowing users to override the default organization name when deploying the devcontainer. The default organization is `vigOS`.

## Related Issue(s)

Closes #33

- **install.sh** (+15 lines): Added `--org ORG` option to argument parsing, help text, and examples. Pass `ORG_NAME` as environment variable to the container.
